# SuperColor 🎨
自分好みの機能だけを詰め込んだ、モダンなデスクトップ用カラーピッカーです。

## ✨ 特徴
* **ワンクリックコピー:** 取得した色（HEX/RGB）を自動でクリップボードに保存
* **履歴機能:** 直近5件の色を履歴として保存・再利用可能
* **最前面固定:** 作業中にウィンドウが隠れない「Always on Top」モード搭載
* **Windows専用:** AIと一緒にPython (CustomTkinter) で開発しました
